Vefsíða

Hönnun og birting vefsíðu með HTML

Um verkefnið

Verkefnislýsing var svohljóðandi:

Veldu eða smíðaðu html sniðmát fyrir vefsíðuna þína, skipulegðu hana með upplýsingum um þína ferilskrá. Skrásettu fyrsta verkefnið og settu á github með því að nota Git bash. Þegar vefsíðan er komin í loftið skal hlekkja hana á vefsíðulista áfangans. Hægt er að gera vefsíðuna á Ensku eða Íslensku.

Enn fremur var tekið fram að við mat á vinnuframlagi væri m.a. litið til eftirfarandi þátta:

  • Vefsíðan er vel skipulögð og auðvelt er að finna allt efni og hönnunar skjöl
  • Nemandinn hefur kynnt sig með ferilskrá og mynd af sér
  • Vinnuframlag og vinnu framkvæmd verkefnis eru gerð góð greinarskil í máli og myndum
  • Allir hlekkir eru til staðar þar sem nemandinn sótti innblástur eða sniðmát. Höfundarverk annarra er getið í texta, ef við á.
  • Ef nemandinn notaði leitarsíðu til að kynna sér frekar efnisorð, þá kemur fram hvaða viðeigandi leitarorð voru notuð.

Undirbúningur

Áður en uppsetning vefsíðunnar hófst þurfti smá undirbúningsvinnu. Ég hafði smá reynslu af notkunn HTML en ég hafði búið til einfalda vefsíðu í forritunaráfanga í menntaskóla. Megnið af því sem hafði lærst þar var því miður gleymt og grafið en þó sem betur fer ekki mjög djúpt. Eftir að hafa horft á nokkur kennslumyndbönd var ég tilbúinn til að hefjast handa.

HTML

Auðvelt var að setja upp vefsíðuna. Það eina sem þurfti að gera var:

  1. Finna sniðmát á html5up.net
  2. Hlaða niður sniðmátinu og setja inn í Brackets
  3. Breyta síðunni með eigin texta og myndum
  4. ???
  5. Gróði

GIT

Til að birta vefsíðuna þurfti einfaldlega að gera eftirfarandi:

  1. Hlaða niður GIT
  2. Búa mér til nýja gagnahirslu (e. repository) á Github
  3. Opna Git bash og nota það til að setja skjölin inn á Github
  4. Nota Github til að birta síðuna

Hafðu samband

Hægt er að nálgast mig á helstu samfélagsmiðlum eða í gegn um síma eða tölvupóst. Einnig er hægt að senda skilaboð í gegn um þessa vefsíðu en þá verða þau send með bréfdúfu til heimilis míns þar sem hundurinn minn tekur á móti þeim og kemur þeim svo áleiðis til mín.
